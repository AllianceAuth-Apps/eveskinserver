Welcome

Welcome to the Eve SKIN server! This is a simple image server for providing SKIN icons for Eve Online types.

This server is open source. You can find the code on GitLab.

Syntax

  • /skin/{license_type_id}/icon[?size=32|64|128|256|512|1024]

Errors

  • HTTP 404 for unknown type IDs
  • Defaults to size = 64 when invalid or no size is given

Examples

  • /skin/34599/icon
  • /skin/47290/icon?size=128
